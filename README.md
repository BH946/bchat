# bchat
**자바 Swing 기반 다중 채팅 프로그램**   
자바 소켓으로 TCP, UDP 통신 + 멀티스레드 + 오라클(오렌지) + MyBatis 경험이 목적

구현 상세 바로가기: **[기능개발](https://github.com/BH946/bchat/blob/main/기능개발.md)**

<br><br>

## Used Skills & Version

- Java 17
- IDE: IntelliJ
- Build Tool: Gradle - Groovy
- Oracle 19 + Oracle Client 11 + Orange 7(dba)
- MyBatis 3.5.7
